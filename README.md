# Подписка для обхода белых списков от zieng2

Все четыре файла имеют один источник и обновляются одновременно.

## Вариации подписки

Рекомендованная версия подписки — **vless_lite.txt**. Она содержит только проверенные подсети, протестированные во время ограничений, и подходит практически для всех клиентов. 

Остальные версии:

+ **vless_universal.txt** — полная версия подписки. Содержит все доступные подсети.

+ **vless_nolite.txt** — используется исключительно для тестирования. Содержит все подсети, кроме проверенно рабочих. Желающие помочь могут скидывать рабочие профили в Issues.

+ **vless.txt** — устаревшая версия подписки, не добавляющаяся в v2rayN. Использовать **не рекомендуется**. В будущем поддержка будет прекращена.

## Советы и интересная информация о подписке

**1.** Рекомендуется глобально включить опцию 'Разрешить небезопасные' в своем клиенте.

**2.** По отзывам пользователей, в зависимости от клиента могут отличаться рабочие сервера. Попробуйте разные клиенты.

**3.** Источниками серверов выступают различные публичные и непубличные источники. Если у вас есть рабочие конфиги и вы хотите поделиться ими, можете использовать Issues.

**4.** DNS в настройках клиента не должен влиять на работоспособность. Ранее была замечена проблема, что из-за недоступности DNS от Cloudflare / Google / других, конфиги с доменными именами не работали до смены на Яндекс. Те конфиги все еще присутствуют в подписке, но в актуальной версии подписки все они сразу представлены с соответствующими IP-адресами.

**5.** При доступном интернете должны работать вплоть до 100% серверов. Скрипт проверяет работоспособность конфигов при обновлениях. Но, как уже упоминалось выше, прошу использовать ее только при необходимости из-за отключений интернета.
